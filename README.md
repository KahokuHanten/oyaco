### エレベーターピッチ

「親子のコミュニケーションに困っているという課題」を解決したい<br>
「離れて暮らしている親子」向けの<br>
「親子の絆を強くする」ことができる<br>
「OYACO」です。<br>
これは「親子の会話ネタを提供する」ことができ、<br>
「雑誌やテレビから得る情報」とは違って、<br>
「その親子にぴったりのネタをタイムリーに提供してくれる機能」が備わっています。
